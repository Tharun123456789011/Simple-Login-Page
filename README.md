# 🔐 Modern Login Form (HTML & CSS)

A sleek and responsive **Login Form UI** built using pure HTML and CSS.  
This project features a **glassmorphism design**, clean layout, and smooth user experience.

---

## ✨ Features

- 🎨 Modern **Glassmorphism UI**
- 📱 Fully responsive design
- 🔒 Username & Password input fields
- 👁️ Clean and minimal layout
- 💡 Styled placeholders and focus effects
- ✔️ "Remember Me" checkbox
- 🔗 "Forgot Password" link
- 🚀 Beginner-friendly project

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, Transitions, Backdrop Filter)
